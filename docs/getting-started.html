Getting started

Install, build, and ship with confidence.

Overview Getting started GitHub

Install

npm install my-library

Build scripts

  • npm run build – compile ESM, CJS, and bundles.
  • npm run test – execute unit tests.
  • npm run test:browser – verify browser bundle behaviour.

Next steps

Update the exports in src/index.ts, customise documentation under docs/, and configure GitHub Actions to automate releases.
