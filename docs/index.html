Why this template?

This repository bundles everything you need to publish a high-quality npm package: strict TypeScript settings, multiple distribution targets, end-to-end testing, and CI/CD automation.

Key capabilities

  • Builds ESM, CommonJS, and browser bundles out of the box.
  • Automated checks enforce bundle size and verify exports in multiple environments.
  • Smoke tests and documentation deployment keep every release trustworthy.
